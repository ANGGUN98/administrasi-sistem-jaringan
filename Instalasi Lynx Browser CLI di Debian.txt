# apt install lynx -y

Cara menggunakan :
- Cari keywoard Linux di lynx
  # lynx linux
  # lynx google.com
- allow cookie : Y  
- tekan 'q' untuk mengakhiri 


Kelebihan :
- Penggunaan kuota/data yang sedikit karena browser ini tidak menampilkan gambar dan elemen kode seperti html,css,js,cookie,dan lain-lain.
- Browser ini hanya menampilkan text saja.
